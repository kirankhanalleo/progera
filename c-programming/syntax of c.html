Syntax of C

Every programming language has its own syntax. Like any other programming language C Language has its own syntax. Syntax defines how codes must be written and enclosed inside a program.

  • Documentation Section

    • The documentation section is a part of of the program where the programmer writes the details associated with the programs. It may include file name, name of the programmer, date, and description related to the program. It is enclosed inside comment.

    /*
    Name of the program: Hello world.c
    Author Name: progera.com
    Date: 2/10/2022
    */
  • Header/Link Section

    • The link section is a part of the Program where all the header files to be used in the program is declared. The C library provides numerous header files.

    #include <stdio.h>
  • Definition Section

    • The definition section is a part of the Program where a programmer defines constants.

    #define PI 3.14;
    #define name "progera.com";
  • Global Declaration Section

    • The definition section is a part of the Program where a programmer defines constants. The global variable and user defined functions are also declared in here.

    float area(int a)
    int x=10;
  • Main Function Section

    • The main function section contains two part, a declaration part and an execution part. All the variables to be used in the program are declared in declaration part and the execution part begins with curly braces and ends with closed curly braces.

    int main()
    {
        int x=10;
        printf("%d",x);
        return 0;
    }
  • Sub Program Section

    • The user-defined functions are declared and executed here.

    int sum(int a,int b)
    {
        return a+b;
    }
