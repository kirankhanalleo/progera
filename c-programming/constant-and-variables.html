Constants and Variables in C

Variables

Variables in C is a container which contains the value of a particular data type. The value stored in a variable can be altered throughout the program.

int a=10;
char c=a;
a=20;
c=z;

In the above example, initially the value of a is initialized as 10 and the value of c is initialized which is later modified to 20 and z respectively.

Constants

Constant are like variable, except that the value of constants cannot be changed after it is defined. The value remains the same throughout the program. #define is used to define constants in C.

#define PI 3.14;
#define name "progera.com";

The value of PI and name remains the same throughout the program
