C Unary Operators

What is an operator?

Operators are the pre-defined characters or the symbols in the C libarary that performs operations on operands.

Unary Operator

Unary operators are the operators that operate on single operand to change the previous value it holds. There are five unary operators in C:

1. Unary Plus (+)

Unary Plus operator changes the sign of the operands. The negative operand becomes positive and the positive remains positive itself.

int a=-10;
+a;

The value of a changes from -10 to 10.

2. Unary Minus (-)

Unary Minus operator changes the sign of the operands. The positive operand becomes negative and the negative becomes positive.

int a=-10,b=10;
-a; -b;

The value of a changes from -10 to 10 and the value of b changes from 10 to -10.

3. Increment Operator (++)

The increment operator increases the value of the operand by 1.

int a=10;
a++;
//a++ works as same as a=a+1.

The value of a becomes 11.

4. Decrement Operator (--)

The decrement operator decreases the value of the operand by 1.

int a=10;
a--;
//a-- works as same as a=a-1.

The value of a becomes 9.

5. Logical Negation Operator (!)

The logical negation operator complements or reveres the meaning of the operands. Arithmetic operators are implicity converted into bool type. The true value becomes false and vice-versa.

bool a=true;
!(a);

The variable a which is initially true becomes false after using negation operator on it.
